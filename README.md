# 🎵 Harmony Player

A sleek, minimalist music player built with Vue.js and Tailwind CSS. Designed for both desktop and mobile with a clean black interface.

## ✨ Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Playlist Management** - Browse and select from a growing library of songs
- **Audio Controls** - Play, pause, skip, shuffle, and repeat functionality
- **Volume Control** - Adjustable volume slider
- **Progress Bar** - Click or drag to seek through tracks
- **Notifications** - Stay updated with the latest player news
- **Creator Info** - Contact the developer for song suggestions

## 🚀 Live Demo

Open `index.html` in your browser to start listening.
